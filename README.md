# AI Sales Context Resolver Demo

This example simulates a 'context resolver' for an AI sales agent. It demonstrates how past customer interactions can be processed to generate an enriched context, which is then provided to the AI along with the current query. This helps the AI provide more relevant and informed responses by understanding the customer's history, addressing the limitations of AI agents that only process the current message.

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal using `python main.py`.
